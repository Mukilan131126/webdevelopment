WEBPAGE FOR TEXT
we can see examples by using all text tags
  1. Italic - to draw attention to words or phrases in text
  2. emphasize - words or phrases that should be stressed more than surrounding text
  3. del - to mark text that has been removed from a document
  4. insert - to identify text that has been added to a document
  5. sub - defines subscript text, ex:H2o
  6. sup - defines subscript text, ex:22

Various HTML tags can change how text appears on a web page and make the text attrative
